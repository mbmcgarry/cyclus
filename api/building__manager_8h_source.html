00027 /// The BuildingManager class is a managing entity that makes decisions
00028 /// about which objects to build given certain conditions.
00029 ///
00030 /// Specifically, the BuildingManager queries the SupplyDemandManager
00031 /// to determine if there exists unmet demand for a Commodity, and then
00032 /// decides which object(s) amongst that Commodity's producers should be
00033 /// built to meet that demand. This decision takes the form of an
00034 /// integer program:
00035 ///
00036 /// \f[
00037 /// \min \sum_{i=1}^{N}c_i*y_i \\
00038 /// s.t. \sum_{i=1}^{N}\phi_i*y_i \ge \Phi \\
00039 /// n_i \in [0,\infty) \forall i \in I, y_i integer
00040 /// \f]
00041 ///
00042 /// Where y_i is the number of objects of type i to build, c_i is the
00043 /// cost to build the object of type i, \f$\phi_i\f$ is the nameplate

00045 /// the set I corresponds to all producers of a given commodity.
