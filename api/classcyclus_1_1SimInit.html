cyclus::SimInit Class Reference

Detailed Description

Handles initialization of a simulation from the output database.

After calling Init, Restart, or Branch, the initialized Context, Timer, and Recorder can be retrieved.

Warning:
the Init, Restart, and Branch methods are NOT idempotent. Only one simulation should ever be initialized per SimInit object.
the SimInit class manages the memory of the initialized Context, Timer, and Recorder.
